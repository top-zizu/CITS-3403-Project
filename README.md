# CITS-3403 Project

Description of Application:
'Settling debates' website where users post debate prompts and the community votes on whether they agree or disagree. Would also allow comments to discuss and argue. Debate would have an expiry and the majority side would be the winners. We implement a reputation score for users based on how their votes perform, and implement a leaderboard based off of the reputation score.
